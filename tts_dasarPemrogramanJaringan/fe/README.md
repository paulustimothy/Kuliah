### Cara Menjalankan

1. **Pastikan Backend Berjalan**

   ```powershell
   # Di folder be/
   cd \tts_dpj\be
   python app.py
   ```

2. **Buka Frontend**
   - Buka file `index.html` di browser, atau
   - Gunakan Live Server di VS Code, atau
   - Jalankan local server:
   ```powershell
   # Di folder fe/
   cd \tts_dpj\fe
   python -m http.server 8000
   ```
   Kemudian buka `http://localhost:8000`
